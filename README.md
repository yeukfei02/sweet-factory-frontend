# sweet-factory-frontend

sweet-factory-frontend

url: <https://sweet-factory-frontend.vercel.app/>

## Requirement

- install pnpm
- install node (v14+)

## Testing and run

```zsh
// install node dependencies
$ pnpm install

// development
$ pnpm run start

// create build folder
$ pnpm run build
```

open localhost:3000
